#  General Health Query Chatbot

##  Clear Problem Statement and Goal

The goal of this project is to develop an AI-powered General Health Query Chatbot that can respond to user health-related questions using a Large Language Model (LLM). The chatbot provides general medical information and guidance in a simple and user-friendly manner, while clearly stating that it does not replace professional medical advice.

---

##  Tools & Technologies
- Streamlit (Frontend UI)
- Groq API (LLM inference)
- Llama-3.1-8B-Instant Model
- Python (Backend logic)

---

##  Model Used
- Llama-3.1-8B-Instant (via Groq API)

---

##  Code Quality Features
- Modular function for LLM response handling
- Clean separation of UI and logic
- Session state for conversation memory
- Fully commented code for readability
- Secure API key handling via environment variables

---

##  Key Features
- Chat-based health query system
- Real-time AI responses
- Conversation memory
- Medical safety prompt (no diagnosis or prescriptions)
- Simple and interactive UI

---

##  Results & Findings
- The chatbot successfully responds to general health queries.
- It provides clear and understandable answers for users.
- It maintains conversation context effectively.
- It ensures safe responses by avoiding medical diagnosis.

---

##  Deployment
Deployed using Hugging Face Spaces with Streamlit.
